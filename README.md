****# Guess The Number Game 🎯

A desktop-based Number Guessing Game built using Java Swing as part of my internship at Oasis Infobyte.

## 📌 Overview

This application generates a random number between 1 and 100, and the user has to guess the correct number. The game provides hints whether the entered number is too high or too low and tracks the total number of guesses along with the best score.

## 🚀 Features

- Random number generation between 1 and 100
- Interactive GUI using Java Swing
- Real-time hints (Too High / Too Low)
- Best score tracking
- Total guess counter
- Give Up functionality
- Play Again option
- User-friendly interface

## 🛠️ Technologies Used

- Java
- Java Swing
- AWT Event Handling


## 🎮 How to Play

1. Run the application.
2. Enter a number between 1 and 100.
3. Click **Guess**.
4. The game will tell you whether your guess is too high or too low.
5. Continue guessing until you find the correct number.
6. Use **Play Again to start a new game.
7. Use **Give Up** to reveal the answer.
